## BASIC RULES

### commit rules

"대분류 : 내용" 형식으로 커밋하기

```
디자인 : 변경한 내용~~
기능 : 기능 구현한 내용~~
```

### naming rules

-  \*.vue 파일은 대문자로 시작
-  마이페이지, 컨텐츠 뷰, 컨텐츠 제작 등의 페이지 = views
-  내부에 들어가는 기능단(컴포넌트 요소들) = components
